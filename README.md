# Tides of Tomorrow — Trainer

A standalone trainer interface for **Tides of Tomorrow**.  
Single `.exe`, no dependencies required.

## Features

- Survival & Resources cheats (Health, Ozen, Ammo, Stamina, Oxygen, Fuel...)
- Movement & Exploration (Speed, Fly, No Clip, Teleport, Freeze Time...)
- Dialogue, Story-Link & Special options
- Clean ocean-themed ImGui UI
- Splash screen with loading progress

## Usage

1. Download `TidesOfTomorrowTrainer.exe`
2. Launch the game
3. Run the trainer — no installation needed

## Requirements

- Windows 10 / 11 (x64)
- Game: Tides of Tomorrow

## Build from Source

```bash
git clone https://github.com/USERNAME/REPO.git
cd REPO
mkdir build && cd build
cmake .. -G "Visual Studio 18 2026" -A x64
cmake --build . --config Release
```
